Игра "Лыжник"
Прыжок работает только в крайних положениях
Не стоит ловить снежный ком
Не стоит падать вниз
Стоит ловить подарки
Можно  продолжить игру за 3 подарка при смерти героя
Для принятия решений - Enter
Для отказа - Ecs
Для выхода в меню - Esc
Скин изменится только после перезапуска игры
